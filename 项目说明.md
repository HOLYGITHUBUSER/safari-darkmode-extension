# Safari 全局暗黑模式扩展 - 项目说明

## 项目概述

这是一个为 Safari 浏览器开发的全局暗黑模式扩展，可以为所有网页应用暗黑效果。

## 文件夹结构说明

```
safari-darkmode-项目/
├── 扩展源代码/                    # 扩展的完整源代码
│   ├── manifest.json             # 扩展配置文件（必需）
│   ├── content.js                # 内容脚本（注入到网页中）
│   ├── darkmode.css              # 暗黑模式样式表
│   ├── popup.html                # 弹出界面 HTML
│   ├── popup.js                  # 弹出界面逻辑
│   ├── icon16.png                # 16x16 像素图标
│   ├── icon48.png                # 48x48 像素图标
│   ├── icon128.png               # 128x128 像素图标
│   └── README.md                 # 扩展详细说明
├── SafariDarkMode-Extension.zip  # 分发包（压缩的扩展源代码）
├── 分发说明.md                    # 给接收者的安装说明
└── 项目说明.md                    # 本文件
```

## 文件详细说明

### 扩展源代码/ 文件夹

**manifest.json**
- 扩展的配置文件
- 定义扩展名称、版本、权限等
- 告诉 Safari 如何加载扩展

**content.js**
- 内容脚本
- 在每个网页加载时自动注入
- 负责检查暗黑模式状态并应用样式

**darkmode.css**
- 暗黑模式的 CSS 样式
- 使用 CSS filter 技术反转颜色
- 包含对图片、视频等媒体的特殊处理

**popup.html**
- 扩展弹出界面的 HTML 文件
- 显示开关按钮
- 用户点击扩展图标时显示

**popup.js**
- 弹出界面的 JavaScript 逻辑
- 处理开关切换
- 保存用户设置到浏览器存储

**icon16.png / icon48.png / icon128.png**
- 扩展图标
- 不同尺寸用于不同场景（工具栏、扩展列表等）

**README.md**
- 扩展的技术文档
- 包含功能说明、技术实现细节

### 分发文件

**SafariDarkMode-Extension.zip**
- 可分发的压缩包
- 包含完整的扩展源代码
- 发送给其他用户使用

**分发说明.md**
- 给接收者的安装指南
- 详细的步骤说明
- 系统要求和注意事项

## 如何使用

### 本地安装（你自己使用）
1. 打开 Safari > 设置 > 扩展
2. 启用开发人员模式
3. 加载未解包的扩展项
4. 选择 `扩展源代码` 文件夹

### 分发给他人
1. 发送 `SafariDarkMode-Extension.zip` 给对方
2. 同时发送 `分发说明.md`
3. 对方按照说明安装即可

## 技术栈

- **Manifest V2**: Safari 兼容的扩展格式
- **JavaScript**: 内容脚本和弹出界面逻辑
- **CSS**: 暗黑模式样式
- **Chrome Storage API**: 保存用户设置

## 功能特性

- ✅ 全局暗黑模式，对所有网页生效
- ✅ 一键开关，方便切换
- ✅ 自动保存设置
- ✅ 使用 CSS filter 技术，兼容性好
- ✅ 图片、视频等媒体自动保持正常显示
