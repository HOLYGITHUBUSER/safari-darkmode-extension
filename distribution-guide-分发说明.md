# Safari 全局暗黑模式扩展 - 分发说明

## ⚠️ 重要提示

**在 Safari 中加载扩展时，请选择 `e-mplefaafa-irdarkmode` 文件夹（解压后的文件夹）**

## 文件说明

**SafariDarkMode-Extension.zip** - 扩展源代码压缩包

## 安装方法（接收者）

### 第一步：解压文件
```bash
unzip SafariDarkMode-Extension.zip
```
或直接双击解压

### 第二步：在 Safari 中加载
1. 打开 Safari 浏览器
2. 菜单栏选择：Safari > 设置 > 扩展
3. 勾选"在菜单栏显示"复选框
4. 勾选"开发人员模式"复选框
5. 点击"加载未解包的扩展项"按钮
6. **选择解压后的 `extension-source-扩展源代码` 文件夹** ⚠️
7. 在扩展列表中勾选"全局暗黑模式"

### 第三步：使用
1. 点击 Safari 工具栏的扩展图标（拼图形状）
2. 找到"全局暗黑模式"并点击
3. 在弹出的面板中切换开关即可启用/禁用暗黑模式
4. 设置会自动保存，下次访问时生效

## 系统要求
- macOS 11.0 或更高版本
- Safari 14.0 或更高版本

## 功能特性
- 全局暗黑模式，对所有网页生效
- 一键开关，方便切换
- 自动保存设置
- 使用 CSS filter 技术，兼容性好

## 注意事项
- 此扩展使用 CSS filter 技术，可能会影响某些网站的颜色显示
- 图片、视频等媒体元素会自动反转以保持正常显示

## 技术支持
如遇问题，请检查：
1. Safari 版本是否符合要求（macOS 11.0+，Safari 14.0+）
2. 是否正确启用了开发人员模式
3. 是否选择了正确的文件夹
4. 扩展是否在扩展列表中勾选启用

## 常见问题
**Q: 扩展显示为"临时"状态？**
A: 这是正常的，因为是通过开发者模式加载的未签名扩展。功能完全正常。

**Q: 重启 Safari 后扩展失效？**
A: 开发者模式加载的扩展在重启后可能需要重新启用，这是 Safari 的安全机制。
