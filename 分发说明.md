# Safari 全局暗黑模式扩展 - 分发说明

## 文件说明

**SafariDarkMode-Extension.zip** - 扩展源代码压缩包

## 安装方法（接收者）

### 第一步：解压文件
```bash
unzip SafariDarkMode-Extension.zip
```
或直接双击解压

### 第二步：在 Safari 中加载
1. 打开 Safari
2. 菜单栏：Safari > 设置 > 扩展
3. 勾选"在菜单栏显示"
4. 勾选"开发人员模式"
5. 点击"加载未解包的扩展项"
6. 选择解压后的 `simple-safari-darkmode` 文件夹
7. 在扩展列表中勾选"全局暗黑模式"

### 第三步：使用
点击 Safari 工具栏的扩展图标，切换开关即可启用/禁用暗黑模式。

## 系统要求
- macOS 11.0 或更高版本
- Safari 14.0 或更高版本

## 功能特性
- 全局暗黑模式，对所有网页生效
- 一键开关，方便切换
- 自动保存设置
- 使用 CSS filter 技术，兼容性好

## 注意事项
- 此扩展使用 CSS filter 技术，可能会影响某些网站的颜色显示
- 图片、视频等媒体元素会自动反转以保持正常显示

## 技术支持
如遇问题，请检查：
1. Safari 版本是否符合要求
2. 是否正确启用了开发人员模式
3. 是否选择了正确的文件夹
